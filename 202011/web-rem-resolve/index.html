REM 解决方案

主要思路

通过响应式设计稿尺寸不一的情况下，通过 resize事件 时判断窗口选用对应客户端的 设计稿宽度 运算，辅以 @media响应式 即可完美复现响应式的设计啦。

方案

以需开发 PC端 和 手机端 的响应式项目为例

  • 将js代码放在 <head> 内，且不依赖任何插件以保证他最早被执行。

  • css预处理器编写函数(我这边用的是stylus)
